| ---- | ---- |
| Swagger 界面 | 启动 swagger 界面 |
| 通用 API | 启动通用 API |
| RK TV | 启动 RK TV |
| Prometheus 客户端 | 启动 prometheus 客户端 |
| 日志拦截器 | 启动日志拦截器 |
| 监控拦截器 | 启动监控拦截器 |
| 元数据拦截器 | 启动元数据拦截器 |
| 调用链拦截器 | 启动调用链拦截器 |
| 权限拦截器 | 启动 basic & ApiKey 权限拦截器 |
